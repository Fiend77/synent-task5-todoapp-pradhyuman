# To-Do Website

A simple, responsive, and modern To-Do List web application built using **HTML**, **CSS**, and **JavaScript**. This application allows users to manage daily tasks efficiently with an intuitive, glassmorphism-styled interface.

## Features

* **Task Management:** Add new tasks and easily delete them.
* **Completion Tracking:** Mark tasks as completed with a custom toggle and visual strikethrough.
* **Visual Progress Bar:** Real-time percentage tracking of completed vs. total tasks.
* **Dynamic Greeting:** Time-based greeting that updates automatically based on the user's local clock.
* **Data Persistence:** Save tasks using the browser's Local Storage API so data is not lost on refresh.
* **Responsive Design:** Clean, modern interface that looks great on both desktop and mobile devices.

## Technologies Used

* HTML5
* CSS3 (Glassmorphism styling)
* JavaScript (ES6)
* Local Storage API
